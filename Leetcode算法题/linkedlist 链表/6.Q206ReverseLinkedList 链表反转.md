**Q206ReverseLinkedList 链表反转**

**方法一：使用双指针，循环将每个节点的next指向前一个节点**

```java
public ListNode reverseList(ListNode head) {
    ListNode prev = null;
    ListNode cur = head;
    while (cur != null) {
        ListNode temp = cur.next;
        cur.next = prev;
        prev = cur;
        cur = temp;
    }
    return prev;
}
```
